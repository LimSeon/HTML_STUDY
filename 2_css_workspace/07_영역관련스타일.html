여백 및 간격 관련 스타일

기준 div

기준 div

영역 구분 (바깥쪽에서 안쪽으로) 1. margin : 바깥쪽 여백 영역
2. border : 테두리 영역 => 여기까지가 실제 요소의 크기
3. padding : 내용물 영역이랑 테두리 영역 사이의 거리(안쪽여백)
4. content : 실제 내용물이 들어있는 영역
실제 요소의 크기(영역) == content + padding + border

padding

내용물 영역(content)와 테두리 영역(border) 사이의 거리(여백)을 조절하는 스타일 속성
