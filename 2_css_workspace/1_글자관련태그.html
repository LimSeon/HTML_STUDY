문단을 나누는 태그

문단 영역을 나누는 태그로는 p태그와 pre태그가 있다.
단, p태그는 줄바꿈 입력을 별도의 태그로 지정해야한다.
그리고 공백은 한개의 공백만 표시하기 때문에         기호문구를 작성해야한다. 

       pre태그는 여러         공백이 인식 가능하고
       줄바꿈 등을 포함해서 입력한 내용 그대로를 표현하는 태그이다.

그 밖에 글자를 표현해주는 태그들!

일반 글꼴(태그로 감싸지지 않은 텍스트)

b: 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그

em : 글자를 기울여서 표시하는 태그

i : 글자를 기울여서 표시하는 태그

s : 글자에 취소선을 넣어주는 태그

mark : 형광펜 효과를 주는 태그들

small : 글자를 작게 표현해주는 태그

big : 글자를 크게 표현해주는 태그

sub : 기본 글자에 아래첨자를 나타내는 태그

sup : 위첨자를 나타내는 태그

글자 관련 태그 응용 문제

태그들은 해당 태그 내에서 중첩으로 사용가능하다.
굵은 글자를 사용할 수도 있고
기울이거나, 취소선을 넣는 등 다양하게 사용할 수 있다.
형광펜을 넣어 글자를 강조할수도 있다
